# 🚦 GATEWAY — 합류 관문

세 레인, 게이트는 하나. 어느 줄을 열고, 언제 옮길까?
**머리(실시간 우선순위 판단)와 손(타이밍)이 동시에 바빠지는** 모바일 원터치 관제 게임.

▶ **플레이**: https://junhee1219.github.io/gateway/

## 규칙

- 세 레인으로 에너지 오브가 떨어져 쌓인다. 레인이 가득 차 넘치면 게임오버.
- **레인을 탭**하면 게이트가 그 레인으로 이동해 바닥부터 배출한다.
- **같은 레인을 연타하면 펌핑** — 배출이 빨라진다. 손은 쉬지 않는다.
- 큰 오브일수록 점수가 높지만(100/300/800) 통과가 느리다 — "제일 찬 줄부터"가 항상 정답이 아니다.
- **골드 오브**는 5배 점수, 단 착지 후 5초 안에 못 빼내면 증발한다.
- 게이트 이동에는 시간이 들고, 배출 중 옮기면 진행도가 날아간다 (전환 페널티).
- 80% 이상 찬 레인에서 배출을 시작하면 **구사일생 ×2**.
- 연속 배출 콤보 (5콤보 ×1.5 / 10콤보 ×2 / 20콤보 ×3).

## 구조

| 파일 | 역할 |
|---|---|
| `core.js` | 순수 게임 로직 (DOM 없음) — 스폰/낙하/배출/콤보/오버플로 |
| `game.js` | Canvas 렌더러 + 입력 + 이펙트 (파티클/셰이크/슬로모/비트 펄스) |
| `audio.js` | Web Audio 합성 — 위험도 적응형 신스웨이브 BGM + SFX (외부 에셋 없음) |
| `test.js` | 로직 테스트 + 봇 3종 밸런스 시뮬레이션 — `node test.js` |
| `sw.js` `manifest.json` | PWA (오프라인 + 홈 화면 설치) |
| `icon.html` `og-card.html` | 아이콘/OG 카드 생성기 (배포 불필요) |

## 개발 메모

- 배포 시 `sw.js`의 `CACHE` 버전을 올려야 클라이언트가 갱신된다.
- 오디오는 첫 사용자 제스처에서 unlock (iOS 필수) — `GatewayAudio.init()`.
- 밸런스 철학: 방치 봇은 1분 내 사망, 탐욕 봇(제일 찬 줄)은 스마트 봇보다 ~20% 일찍 죽는다.
  전략이 보상받되 아무도 영생할 수 없다. 검증: `node test.js`.
